**Electronics assembly:**

```
Step 1: Solder LEDs to controller board
  - 6 × WS2812B LEDs
  - Arranged in hexagonal pattern (60° spacing)
  - Chain: GPIO 2 → LED1 → LED2 → ... → LED6
  - Power: 3.3V, GND (shared across all LEDs)

Step 2: Program ESP32
  - Install Arduino IDE
  - Add ESP32 board support
  - Upload code (provided in repo: strobing.ino)
  - Test: LEDs should strobe at 1.09375 Hz

Step 3: Mount LEDs in frame
  - Insert LEDs into front-bridge piece (6 hexagonal holes)
  - Hot-glue in place (secure, waterproof)
  - Orient LEDs inward (toward eyes when worn)

Step 4: Attach controller + battery
  - Controller board: Velcro to right temple piece
  - Battery: Velcro to left temple piece
  - Wires: Route through temple pieces (hidden)
  - Power switch: Accessible on right temple

Step 5: Final assembly
  - Snap front-bridge onto left/right temple pieces
  - Adjust nose pads for comfort
  - Test fit, adjust as needed
```

**Code (strobing.ino):**

```cpp
#include <Adafruit_NeoPixel.h>

#define LED_PIN 2
#define NUM_LEDS 6

Adafruit_NeoPixel strip(NUM_LEDS, LED_PIN, NEO_GRB + NEO_KHZ800);

float frequency = 1.09375; // Hz, substrate sub-harmonic
float brightness = 128; // 0-255, adjustable via potentiometer

void setup() {
  strip.begin();
  strip.show(); // Initialize all pixels to 'off'
}

void loop() {
  float t = millis() / 1000.0; // Time in seconds
  float phase = 2 * PI * frequency * t;
  float intensity = brightness * (0.5 + 0.5 * sin(phase)); // Sine wave

  for (int i = 0; i < NUM_LEDS; i++) {
    strip.setPixelColor(i, strip.Color(intensity, intensity * 0.9, intensity * 0.8)); // Warm white
  }
  strip.show();
  delay(10); // 100 Hz update rate
}
```
