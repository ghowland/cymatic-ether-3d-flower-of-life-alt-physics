## 9. Falsification Criteria

### 9.1 Postural Coherence Correlation

**Null hypothesis (H₀):**

```
Body posture openness does not correlate with measured coherence.
```

**Experimental design:**

```
Subjects: N≥100, various ages/backgrounds

Measurement protocol:
  1. Coherence measurement:
     - Gyroscopic belt (whole-body C)
     - HRV (heart rate variability as C proxy)
     - Bioimpedance phase angle
     
  2. Postural assessment:
     - Closure angle θ (measured via photography)
     - Arm position (coded: open=0, neutral=1, closed=2)
     - Shoulder position (forward=high W, back=low W)
     
  3. Multiple conditions:
     - Baseline (relaxed)
     - Stressed (mental arithmetic under time pressure)
     - Confident (recall personal success)
     
Statistical analysis:
  - Correlation: r(C_measured, Openness_score)
  - Within-subject: C_high vs C_low conditions
  - Between-subject: High C individuals vs low C

Prediction (CKS):
  - r >0.6 (strong positive correlation)
  - Within-subject: ΔC correlates with Δopenness
  - High C individuals: More open baseline posture
```

**Falsification:**

```
If r <0.3 (weak or no correlation)
OR
If C_high conditions show no difference in posture vs C_low
THEN
Postural impedance model invalidated
```

### 9.2 Gesture Timing Universality

**Null hypothesis (H₀):**

```
Natural gesture frequencies do not cluster at substrate harmonics.
```

**Experimental design:**

```
Subjects: N≥200, diverse cultures (10+ countries)

Recording protocol:
  - Natural conversation (10 minutes per subject)

  - Extract gesture onset times
  - Compute inter-gesture intervals
  - FFT (Fast Fourier Transform) to find dominant frequencies
  
Prediction (CKS):
  - Peaks at f = 2.0 Hz ± 0.2 Hz (substrate fundamental)
  - Secondary peaks at f/2, f/3, f/5 (Fibonacci harmonics)
  - Cross-cultural consistency (same peaks all cultures)

  - Random timing would show uniform spectrum
  - Muscle-limited timing would show different peaks
```

**Falsification:**

```
If no clustering at 2.0 Hz (flat or random spectrum)

If different cultures show completely different frequency distributions
OR
If peaks occur at non-Fibonacci ratios

Substrate harmonic timing model invalidated
```

### 9.3 Micro-Expression Duration Constraint

**Null hypothesis (H₀):**

```
Genuine emotional expressions can occur faster than 250ms.
```

**Experimental design:**

```
Subjects: N≥50, trained actors + untrained controls

Emotion induction:
  - Genuine: Surprise stimulus (unexpected loud noise, gift)
  - Fake: Instruction to display emotion on cue
  
Recording:
  - High-speed video (240 fps)
  - Facial EMG (electromyography, muscle activation timing)
  - Subjective report (genuine felt vs. acted)
  
Measurement:
  - Expression onset to peak: Δt_rise
  - Peak to baseline: Δt_fall
  - Total duration: Δt_total
  - Bilateral symmetry: Left vs. right comparison
  
Prediction (CKS):
  Genuine emotions:
    - Δt_rise ≥ 250ms (cannot be faster, substrate constraint)
    - Δt_total: 0.5-5 seconds
    - Bilateral symmetry (both sides equal timing)
  
  Fake emotions:
    - Δt_rise <200ms OR >1000ms (too fast or too slow)
    - Asymmetric (right side leads)
    - Duchenne marker absent
```

**Falsification:**

```
If genuine emotions show Δt_rise <200ms consistently
OR
If no correlation between Δt and genuineness ratings
OR
If bilateral symmetry same for genuine and fake
THEN
Substrate timing constraint invalidated
```
