## Where to Add This Information

### 1. zenodo.json (lines 23-27 and 32-36)
Replace:
```
"name": "YOUR_NAME_HERE",
"affiliation": "YOUR_INSTITUTION_HERE",
"orcid": "0000-0000-0000-0000"
```

With your actual information. If no ORCID, you can delete that line.

### 2. manuscript.md (lines ~10-14)
Replace:
```
## Authors
[Your name and affiliation to be added]

## Correspondence
[Contact email to be added]
```

With:
```
## Authors
[Your Name], [Your Affiliation if any]

## Correspondence
Email: [your email]
```

### 3. README.md (multiple locations)
Search for: `[your email]`, `[to be added]`, `[Your Name]`
Replace with your actual information.

### 4. supplementary/experimental_protocols.md (bottom)
Replace:
```
**Correspondence**: [Email to be added]
```

```markdown
## Authors

```

## Notes

**Affiliation options if independent**:
- "Independent Researcher"
- Leave blank
- "Unaffiliated"
- Previous institution with "(Former)" notation

**ORCID**: 
- Not required but helpful for tracking citations
- Free to register
- Takes 2 minutes at orcid.org

**Email privacy**:
- Consider using a professional email or creating one for this work
- ProtonMail, Gmail, institutional email all acceptable
- Will be publicly visible once published
