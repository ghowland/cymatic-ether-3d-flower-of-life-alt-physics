```
Γ = (Z_mortar - Z_brick) / (Z_mortar + Z_brick)

```

## 10. Troubleshooting and Optimization

### 10.1 Common Issues

**Problem: Foundation vibration test fails**

**Possible causes:**
```
- Voids in concrete (air pockets)
- Poor ground rod connection
- Insufficient compaction
- Wrong aggregate (limestone not granite)
```

**Solutions:**
```
- Drill test holes, inject grout if voids found
- Re-weld ground connections (exothermic)
- Add auxiliary ground rods, parallel connection
- If severe: May need to pour new foundation
```

**Problem: Brick acoustic test inconsistent**

**Possible causes:**
```
- Mixed batch from multiple kilns
- Inadequate firing (not vitrified)
- High moisture content
- Internal cracks (not visible)
```

**Solutions:**
```
- Test larger sample size (20-30 bricks)
- Calculate standard deviation (should be <15%)
- If high variation: Reject batch, source new supplier
- Dry suspicious bricks in oven, retest
```

**Problem: Mortar joints showing cracks**

**Possible causes:**
```
- Too much water in mix (weak)
- Insufficient curing (dried too fast)
- Wrong lime type (Portland substituted)
- Thermal expansion mismatch
```

**Solutions:**
```
- Re-point joints with correct mortar
- Keep new work damp during curing
- Verify lime type before mixing
- Add copper mesh if not present (flexibility)
```

**Problem: Trident alignment drift**

**Possible causes:**
```
- Wind load (guy wires needed)
- Thermal expansion
- Settling foundation
- Improper initial alignment
```

**Solutions:**
```
- Install guy wires if height >4m
- Verify foundation stability
```
