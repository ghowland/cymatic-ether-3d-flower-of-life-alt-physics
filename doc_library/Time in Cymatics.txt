## Part 7: Time in Different Reference Frames

```

- Counts oscillations in rest frame
- Invariant (all observers agree on count)

- Time measured by external observer
- Depends on reference frame
- Not invariant (different observers disagree)

Relationship:
dτ = √(1 - v²/c²) dt = dt/γ

Proper time always shorter:
τ ≤ t (for any motion)
```
