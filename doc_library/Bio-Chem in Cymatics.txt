## Part 2: The Central Dogma as Pattern Flow

### DNA → RNA → Protein as Pattern Propagation

**DNA: 1D Pattern Template**

**Structure:**
```
Double helix: Two anti-parallel strands
Base pairs: A-T (2 H-bonds), G-C (3 H-bonds)
Backbone: Sugar-phosphate (charged, hydrophilic)
Interior: Bases (hydrophobic, π-stacked)

Dimensions:
- Diameter: 2 nm
- Pitch: 3.4 nm (one full turn, 10 base pairs)
- Base spacing: 0.34 nm along axis
```

**Vibrational modes:**
```
Base pair breathing (H-bond oscillation):
  - A-T: 50-70 THz (weaker, 2 bonds)
  - G-C: 80-100 THz (stronger, 3 bonds)
  
Backbone vibrations:
  - Phosphate: 30-40 THz
  - Sugar ring: 40-60 THz
  
Helical modes:
  - Twisting: 0.1-1 THz (low frequency, collective)
  - Bending: 0.01-0.1 THz (even lower)
  
π-stacking oscillations:
  - Between bases: 3-10 THz (dispersive forces)
```

---

**Cymatic interpretation of DNA:**

**DNA is 1D pattern with 3D embedding:**

```
Primary pattern: Base sequence (A, T, G, C)
  - 4-letter code
  - Digital information (symbolic)
  
Secondary pattern: H-bond network
  - Complementary pairing (A-T, G-C)

Tertiary pattern: Helical geometry

  - 3D standing wave in space
  
Quaternary pattern: Supercoiling, chromatin

  - 4D pattern (geometry + temporal dynamics)
```

**Information encoding:**

```
Not in individual bases (static)
But in:
  - Sequence patterns (recognition sites)
  - Vibrational signatures (A-T vs G-C frequencies)
  - Helical parameters (twist, bend, groove width)
  - Supercoiling state (tension/relaxation)

DNA is 4D barcode:
  - Read spatially (sequence)
  - Read temporally (breathing dynamics)
  - Read vibrationally (H-bond frequencies)
  - Read geometrically (3D shape)
```

### Transcription: Pattern Copying with Transformation

**RNA polymerase as pattern reader:**

**Structure:**
```
Large enzyme complex (~500 kDa)
Multiple domains:
  - DNA binding cleft (recognizes promoter)
  - Active site channel (synthesizes RNA)
  - Exit channel (releases RNA)
  
Size: ~15 nm diameter
Contains ~5000 amino acids
Multiple metal ions (Mg²⁺) in active site
```

**The transcription process in 4D:**

**Step 1: Promoter recognition (spatial pattern matching)**

```
RNA polymerase scans DNA:
  - Slides along double helix (1D diffusion)
  - Probes major groove (reads sequence pattern)
  - Detects promoter sequence (specific pattern)

Promoter (e.g., bacterial -10 and -35 boxes):
  - Specific base sequence
  - Specific vibrational signature
  - Specific groove geometry
  
Recognition mechanism:
  - Protein domains make H-bonds with bases
  - Geometry matches (complementary 3D shapes)
  - Vibrational resonance (domain oscillations match DNA modes)

This is 3D + frequency pattern matching:
  - Must match spatial geometry AND
  - Vibrational modes must couple AND
  - Electrostatic pattern must align
```

**Step 2: DNA melting (pattern opening)**

```
At promoter:
  - RNA polymerase inserts "wedge" domain
  - Separates two DNA strands
  - Opens ~10-base-pair "bubble"

Energetics:
  - Breaking base pairs costs energy (~20-30 kJ/mol each)
  - Provided by conformational change of polymerase
  - Polymerase switches from "open" to "closed" state

Vibrational mechanism:
  - Polymerase domain vibrations (0.1-1 THz)
  - Couple to DNA base pair breathing modes (50-100 THz)
  - Resonant energy transfer
  - Multiple low-frequency phonons → One high-frequency bond break
  - Harmonic up-conversion

This is vibrational catalysis:
  - Enzyme collective modes
  - Pump energy into base pairs
  - Phase-matched energy transfer
  - Like FWM in DWDM: ω₁ + ω₂ + ω₃ → ω_break
```

**Step 3: RNA synthesis (pattern copying with modification)**

```
Template DNA strand exposed:
  - Bases accessible to pairing

NTP (nucleotide triphosphate) substrates:
  - ATP, UTP, GTP, CTP
  - Each has specific geometry and vibrational signature

Polymerase active site:
  - Brings template base and NTP together
  - Forces complementary pairing (A-U, G-C)
  - Geometry constrained by enzyme (spatial phase-matching)

Base pairing:
  - Template base oscillating (H-bond modes)
  - Incoming NTP oscillating
  - When frequencies and phases match:
    → H-bonds form
    → Resonant coupling
    → Stable base pair

Phosphodiester bond formation:
  - NTP has three phosphates (α, β, γ)
  - Bond forms between RNA 3'-OH and NTP α-phosphate
  - Pyrophosphate (PPᵢ = β + γ phosphates) released
  
Catalytic mechanism:
  - Two Mg²⁺ ions in active site
  - Coordinate nucleophile (3'-OH) and leaving group (PPᵢ)
  - Reduce activation energy by electrostatic stabilization
  - Vibrational modes of Mg-O bonds couple to reaction
  
Energy: Exergonic (releases energy)
  - Breaking P-O bonds: Costs ~30 kJ/mol
  - Forming new P-O bond: Releases ~35 kJ/mol
  - Net: -5 kJ/mol (favorable)
  - PPᵢ hydrolysis: Additional -33 kJ/mol
  - Total: -38 kJ/mol (highly favorable, irreversible)
```

**Step 4: Translocation (pattern advancement)**

```
After adding one nucleotide:
  - Polymerase must move forward one base
  - "Ratchet" mechanism

Conformational change:
  - Binding of NTP → Trigger domain closure
  - Domain closure → Force translocation
  - Like Brownian ratchet (biased random walk)

Vibrational picture:
  - Domain oscillations couple to DNA helical modes
  - Phase-locked stepping
  - Each NTP addition → One step forward
  - Frequency: ~50 nucleotides/second
  - Period: ~20 ms per step

This is catalytic cycle oscillation:
  - Bind → Close → Catalyze → Translocate → Open → (Repeat)
  - Limit cycle in conformational space
  - Driven by NTP binding energy
```

**Step 5: RNA release (pattern separation)**

```
At terminator sequence:
  - Specific DNA pattern causes polymerase dissociation
  - RNA released
  - DNA strands re-anneal

Mechanism:
  - Terminator forms hairpin in RNA (stem-loop)
  - Hairpin stabilized by self-complementarity
  - U-rich region after hairpin (weak A-U bonds)

Hairpin formation:
  - RNA folds back on itself (3D pattern)
  - Forms stem (base-paired) and loop
  - Stiff structure

Destabilization:
  - Hairpin pushes against exit channel
  - Mechanical force on polymerase-DNA complex
  - U-A bonds weak (only 2 H-bonds)
  - RNA separates from DNA
  - Polymerase releases

This is force-induced decoherence:
  - Pattern coupling (RNA-DNA) broken
  - Mechanical force from 3D hairpin geometry
  - Weak vibrational coupling (U-A) insufficient to resist
```

### Translation: Pattern to 3D Structure

**mRNA: 1D Pattern Carrier**

```
Single-stranded RNA
Contains:
  - 5' UTR (untranslated region)
  - Start codon (AUG)
  - Coding sequence (triplet codons)
  - Stop codon (UAA, UAG, UGA)
  - 3' UTR

Codons: 3-base units
  - 64 possible (4³)
  - Encode 20 amino acids + stop
  - Redundancy (degeneracy)

Vibrational character:
  - Similar to DNA but single-stranded
  - More flexible (no complementary strand constraint)
  - Can form local secondary structures (hairpins, loops)
```

---

**Ribosome: Pattern-to-Structure Converter**

**Structure:**
```
Two subunits:
  - Small (30S in bacteria): Decodes mRNA
  - Large (50S in bacteria): Catalyzes peptide bonds
  - Together: 70S ribosome (~2.5 MDa)

Composition:
  - 2/3 RNA (rRNA)
  - 1/3 Protein
  - RNA does catalysis (ribozyme)

Key sites:
  - A site (aminoacyl): Incoming tRNA
  - P site (peptidyl): Growing peptide chain
  - E site (exit): Departing tRNA
```

**The translation process in 4D:**

**Step 1: Initiation (pattern recognition)**

```
Small subunit binds mRNA:
  - Recognizes Shine-Dalgarno sequence (bacteria)
  - Or 5' cap structure (eukaryotes)
  - Specific RNA pattern

Start codon recognition:
  - AUG (methionine)
  - Initiator tRNA recognizes AUG
  - Codon-anticodon pairing (complementary)

Assembly:
  - Large subunit joins
  - Initiator tRNA in P site
  - Ready to start
```

**Step 2: Elongation (iterative pattern synthesis)**

```
tRNA selection (A site):
  - Many tRNAs in solution
  - One correct tRNA for each codon
  - How does ribosome select?

Selection mechanism (4D pattern matching):

Spatial:
  - Codon exposed in A site
  - tRNA anticodon must fit geometrically
  - Only correct tRNA has complementary shape

Vibrational:
  - Codon bases oscillating (H-bond modes)
  - Correct tRNA anticodon oscillations match
  - Resonant coupling (phase-locked pairing)
  - Incorrect tRNA: Mismatch in frequencies
  
Time:
  - Correct tRNA: Stable binding (~20 ms)
  - Incorrect tRNA: Dissociates quickly (~1 ms)
  - Selection by kinetic proofreading

Proofreading:
  - EF-Tu GTPase checks pairing
  - GTP hydrolysis = timer
  - If still bound after GTP hydrolysis: Accept
  - If dissociated: Reject
  
Accuracy: 1 error per 10,000 codons
  - Very high fidelity
  - Achieved by 4D pattern matching + kinetic proofreading
```

**Peptide bond formation:**

```
Two amino acids in close proximity:
  - Peptidyl-tRNA in P site (growing chain)
  - Aminoacyl-tRNA in A site (new amino acid)

Active site: 23S rRNA (not protein!)
  - RNA catalyzes reaction (ribozyme)
  - Mechanism: Entropy trap + geometry

Reaction:
  Peptide-tRNA(P) + AA-tRNA(A) → Peptide-AA-tRNA(A) + tRNA(P)
  
Mechanism:
  - P-site tRNA ester bond (peptide-tRNA)
  - A-site tRNA amino group attacks
  - Nucleophilic substitution
  - New peptide bond formed
  - P-site tRNA now uncharged

Catalysis (cymatic view):
  
Ribosome active site:
  - Precise geometry (spatial constraint)
  - Positions nucleophile and electrophile
  - Distance: ~3 Å (optimal for reaction)
  - Angle: Correct for orbital overlap
  
Vibrational assistance:
  - rRNA vibrations (0.1-10 THz)
  - Couple to substrate oscillations
  - Reduce barrier by vibrational excitation
  - Transition state stabilized
  
This is 4D phase-matching:
  - Spatial: Geometry perfect
  - Temporal: Oscillations synchronized
  - Frequency: rRNA modes match substrate
  - Phase: Constructive orbital overlap
  
Rate: ~20 amino acids per second
  - Each peptide bond: ~50 ms
  - Very fast, very accurate
```

**Translocation:**

```
After peptide bond:
  - Must move mRNA by 3 bases (one codon)
  - Must move tRNAs: A→P, P→E, E→exit

Mechanism:
  - EF-G GTPase binds
  - GTP hydrolysis provides energy
  - Conformational change in ribosome
  - Ratchet-like motion

Cymatic picture:
  - Ribosome has two major conformational states
  - Oscillates between them (limit cycle)
  - GTP binding → State 1 (pre-translocation)
  - GTP hydrolysis → State 2 (post-translocation)
  - Frequency: ~20 Hz (20 aa/sec)
  
This is macromolecular breathing:
  - Large-scale conformational oscillation
  - Powered by GTP hydrolysis
  - Synchronized to elongation cycle
```

## Part 3: Protein Folding as Energy Landscape Descent

### The Folding Problem

**Levinthal's paradox:**
```
If protein tried every conformation:
  - 100 amino acids
  - 3 conformations per bond (φ, ψ angles)
  - ~200 flexible bonds
  - 3²⁰⁰ ≈ 10⁹⁵ possible conformations

At 1 conformation per picosecond:
  - Would take 10⁸² seconds
  - Age of universe: ~10¹⁷ seconds
  
But proteins fold in microseconds to seconds!

How?
```

**Resolution (cymatic view):**

Protein doesn't search randomly.

**Folding is descent down energy landscape — a 4D pattern finding its stable attractor.**

---

### The Energy Landscape

**Coordinates:**

```
High-dimensional space:
  - Each amino acid: 2 angles (φ, ψ)
  - 100 amino acids: 200 dimensions
  - Energy as function of these 200 angles

Reduced representation (for visualization):
  - Project to 2D
  - Axes: Collective coordinates (e.g., radius of gyration, number of native contacts)
  - Vertical: Energy
```

**Shape:**

```
Funnel-shaped:
  - Top: Unfolded (many states, high entropy, high energy)
  - Sides: Partially folded (intermediate states)
  - Bottom: Native fold (one state, low entropy, low energy)

Roughness:
  - Not smooth funnel
  - Local minima (traps)
  - Barriers between minima
  - But overall downhill trend
```

---

**The folding trajectory (cymatic interpretation):**

**Initial state (unfolded):**
```
Random coil:
  - No fixed structure
  - High conformational entropy
  - Hydrophobic residues exposed to water (unfavorable)
  - Backbone H-bonds unsatisfied
  - Energy: High
  
Vibrational character:
  - All modes excited
  - No coherence
  - Random thermal motion
  - ψ ≈ 0 (no pattern)
```

**Collapse (hydrophobic collapse):**
```
Within microseconds:
  - Hydrophobic residues cluster
  - Driven by water entropy increase
  - Protein becomes more compact
  - Volume decreases ~50%

Mechanism:
  - Water around hydrophobic groups = ordered (low entropy)
  - When hydrophobic groups cluster:
    → Less water-hydrophobic interface
    → Water released (entropy increases)
    → Favorable free energy change
  
This is spontaneous pattern formation:
  - Disorder (unfolded) → Order (collapsed)
  - But entropy increase in environment (water)
  - Overall: Entropy of universe increases
```

**Secondary structure formation:**
```
On ~millisecond timescale:
  - α-helices form
  - β-sheets begin assembling
  - Local structures stabilize

Mechanism (cymatic):
  
α-helix formation:
  - Backbone C=O at position n
  - Backbone N-H at position n+4
  - H-bond forms: C=O···H-N
  - Creates helical turn
  
Vibrational resonance:
  - C=O stretch: 50 THz (amide I)
  - N-H stretch: 100 THz (amide A)
  - When H-bonded: Both modes shift
  - New coupled mode: ~50 THz
  - Multiple H-bonds → Collective mode
  
Helix as standing wave:
  - 3.6 residues per turn
  - Pitch: 5.4 Å
  - Each H-bond adds to pattern
  - Cooperative formation (zipper-like)
  
β-sheet formation:
  - Two or more strands align
  - Backbone H-bonds between strands
  - Can be parallel or antiparallel
  
Vibrational character:
  - Similar to helix (amide modes)
  - But different geometry (extended not coiled)
  - Collective breathing modes across sheet
```

**Tertiary structure assembly:**
```
On ~100 millisecond to second timescale:
  - Secondary structures pack together
  - Side chain interactions form
  - Disulfide bonds (if any)
  - Final 3D structure achieved

Driving forces:
  1. Hydrophobic effect (main driver)
     - Bury hydrophobic residues
     - Core of protein = hydrophobic
  
  2. H-bonds (directional specificity)
     - Backbone H-bonds (secondary structure)
     - Side chain H-bonds (tertiary contacts)
  
  3. Electrostatics (long-range)
     - Salt bridges (charged residues)
     - Dipole-dipole interactions
  
  4. van der Waals (packing)
     - Shape complementarity
     - Close packing (no voids)

Final state:
  - Densely packed core (~75% of max packing)
  - Specific 3D geometry
  - Unique (or nearly so)
```

### Molecular Chaperones: Guided Folding

**Problem:**

Some proteins cannot fold alone:
  - Aggregate (interact with other unfolded proteins)
  - Get trapped in local minima
  - Too slow to fold (kinetic trap)

**Solution: Chaperones**

---

**GroEL/GroES: The Anfinsen Cage**

**Structure:**
```
GroEL:
  - 14 subunits (7 + 7, two rings)
  - Central cavity (~5 nm diameter)
  - Can encapsulate one protein

GroES:
  - 7 subunits (one ring, lid)
  - Caps GroEL cavity
```

**Mechanism (cymatic view):**

**Step 1: Capture**
```
Unfolded protein binds GroEL cavity:
  - Hydrophobic patches on cavity wall
  - Attract hydrophobic regions of unfolded protein
  - Protein captured
```

**Step 2: Encapsulation**
```
ATP binds GroEL:
  - Conformational change
  - Cavity expands and becomes hydrophilic
  - GroES cap binds (seals cavity)
  
Protein now isolated:
  - Cannot aggregate (alone in cage)
  - Hydrophilic environment (like folded state)
  - Forced to fold
```

**Step 3: Folding**
```
Inside cavity (~10 seconds):
  - Protein folds in isolation
  - No aggregation possible
  - Limited conformational space (cage constrains)
  - This guides folding trajectory

Cymatic interpretation:
  - Cage = boundary condition for pattern
  - Restricts accessible states
  - Creates effective energy landscape with fewer local minima
  - Like putting Chladni plate in different geometry
  - Changes allowed patterns
```

**Step 4: Release**
```
ATP hydrolysis:
  - GroES cap released
  - Protein ejected
  - If folded: Stable
  - If not: Recaptured (iterative cycles)
```

**Cymatic principle:**

Chaperone doesn't provide folding information.

**Chaperone provides 4D boundary conditions:**
- Spatial: Confinement (cavity geometry)
- Temporal: Time window (ATP hydrolysis timer)
- Energetic: Hydrophobic→hydrophilic switch
- Pattern: Restricts accessible conformational space

**Result: Folding trajectory guided through energy landscape**

### Enzyme Catalysis Revisited

**Cymatic view:**

---

### Detailed Example: Serine Protease (Chymotrypsin)

**Function:**
Cleaves peptide bonds (proteolysis)

**Active site:**
```
Catalytic triad:
  - Ser195 (serine, nucleophile)
  - His57 (histidine, general base/acid)
  - Asp102 (aspartate, stabilizes His)

Oxyanion hole:
  - Two backbone NH groups
  - Stabilize negative charge on oxygen
```

---

**The catalytic cycle in 4D:**

**Step 1: Substrate binding (4D recognition)**

```
Substrate (protein with peptide bond to cleave):
  - Must have specific sequence (specificity pocket)
  - Aromatic residue before cleavage site (Phe, Trp, Tyr)
  - Fits into S1 pocket (hydrophobic pocket)

Binding:
  - Substrate peptide inserts into active site cleft
  - Aromatic side chain enters S1 pocket
  - Peptide bond to be cleaved positioned near Ser195

Cymatic recognition:

Spatial:
  - Aromatic ring fits S1 pocket geometry (size, shape)
  - Peptide backbone aligns with catalytic triad
  - Distance Ser195-O to substrate carbonyl-C: ~4 Å

Vibrational:
  - Aromatic ring vibrations (50 THz) couple to pocket
  - Peptide C=O vibration (50 THz) couples to Ser-OH
  - Resonant energy transfer begins

Conformational:
  - Enzyme undergoes "induced fit"
  - Two domains close around substrate
  - Traps substrate in optimal geometry
  - Like closing jaws
```

**Step 2: Acylation (first half-reaction)**

```
Mechanism:

Ser195-OH attacks peptide C=O:
  - His57 acts as base (abstracts proton from Ser-OH)
  - Ser-O⁻ (alkoxide) much better nucleophile
  - Attacks carbonyl carbon
  - Tetrahedral intermediate forms

Tetrahedral intermediate:
  - Four groups around carbon: -OR, -NH-, -O⁻, -R
  - Oxygen negative (oxyanion)
  - Stabilized by oxyanion hole (two NH···O H-bonds)

Collapse:
  - C-N bond breaks
  - Amine leaves (first product)
  - Acyl-enzyme intermediate remains

Result:
  - Peptide bond broken
  - Substrate C now bonded to Ser195 (ester bond)
```

**Cymatic mechanism (detailed):**

```
Before proton transfer:
  Ser195-OH vibrating: O-H stretch at ~105 THz
  His57 oscillating: N···H bond fluctuating
  
Proton transfer (quantum tunneling):
  - When Ser-OH and His geometries align (spatial)
  - When vibrational phases match (temporal)
  - Proton probability density extends toward His
  - Tunnels across ~2 Å gap
  - Transfer time: ~10 femtoseconds (faster than vibration!)
  
This is vibrational phase-matched tunneling:
  - O-H stretch elongates OH bond
  - N lone pair stretches toward H
  - When both extended: Transfer occurs
  - Frequency matching essential
  
After transfer:
  Ser195-O⁻ highly reactive
  C=O bond weakened (substrate carbonyl accepting electron density)
  O-C double bond → partial triple bond (resonance)
  
Nucleophilic attack:
  - Ser-O⁻ orbital overlaps with substrate C=O π* orbital
  - When phases align (constructive interference)
  - Electron density flows: O⁻ → C
  - New C-O bond forms
  - Transition state: Tetrahedral carbon
  
Transition state stabilization:
  - Oxyanion develops negative charge
  - Oxyanion hole NH groups donate H-bonds
  - Electrostatic stabilization
  - Lowers energy of transition state by ~40 kJ/mol
  
Vibrational picture:
  - Oxyanion vibrations (C-O⁻ stretch ~40 THz)
  - NH groups vibrate (N-H stretch ~100 THz)
  - Coupled mode emerges: H-bonded complex
  - Collective oscillation stabilizes structure
  
Collapse to acyl-enzyme:
  - C-N bond breaking
  - Amine N lone pair reforms
  - C-N bond vibration (30 THz) amplitude increases
  - Critical amplitude: Bond breaks
  - Amine diffuses away
```

**Step 3: Deacylation (second half-reaction)**

```
Water enters active site:
  - Positioned by His57
  - Attacks acyl-enzyme C=O
  
Same mechanism as acylation:
  - His abstracts proton from H₂O → OH⁻
  - OH⁻ attacks carbonyl
  - Tetrahedral intermediate (again)
  - Collapse: C-O(Ser) bond breaks
  - Carboxylic acid product released
  - Enzyme regenerated
```

**The 4D catalytic orchestration:**

```
DIMENSION       MECHANISM                          ENHANCEMENT
─────────────────────────────────────────────────────────────────────────

Spatial         Substrate positioned precisely     10³-10⁴ fold
                Distance Ser-O to C=O: 3-4 Å       (proximity effect)
                Geometry optimal for attack        
                
Temporal        Vibrational phase synchronization  10²-10³ fold
                Ser-OH, His, substrate in phase    (timing effect)
                Proton transfer when aligned

### Why ATP Is "High Energy"

**Not because P-O bonds are weak!**

```
P-O bond strength: ~360 kJ/mol (normal ester bond)
Breaking P-O: Costs energy (endothermic)

So why is hydrolysis exothermic?
```

**Reason 1: Electrostatic repulsion**

```
Three phosphates = Four negative charges (-4 total)
Jammed together in <1 nm space

Coulomb repulsion energy:
  E ∝ q₁q₂/r
  
  For adjacent phosphates:
    q = -1, -1
    r ≈ 0.3 nm
    E ≈ +80 kJ/mol (repulsive, unfavorable)
  
Total repulsion between all three: ~150-200 kJ/mol

This is pattern strain:
  - Negative charges forced together
  - Want to separate
  - Geometry constrained by bonds
  - Like compressed spring
```

**Reason 2: Resonance stabilization of products**

```
When ATP → ADP + Pi (inorganic phosphate):

Pi (HPO₄²⁻):
  - Can resonate (delocalize electrons)
  - Four P-O bonds equivalent (by resonance)
  - Stabilization: ~25 kJ/mol

ADP:
  - Less repulsion (one fewer phosphate)
  - β-phosphate can resonate better
  - Stabilization: ~25 kJ/mol
  
Total: ~50 kJ/mol stabilization of products
```

**Reason 3: Hydration (solvation)**

```
ATP in water:
  - Tight cluster of charges
  - Water molecules ordered around it
  - Low entropy

Products (ADP + Pi):
  - Charges more separated
  - More water molecules interact
  - Higher entropy
  - Stabilization: ~20 kJ/mol
```

**Total free energy of hydrolysis:**
```
ΔG° = -30.5 kJ/mol (standard conditions)
ΔG in cell ≈ -50 kJ/mol (non-standard, favorable)

Sources:
  Electrostatic relief: -150 kJ/mol
  + Bond breaking: +360 kJ/mol
  + Bond forming (with H₂O): -320 kJ/mol
  + Resonance: -50 kJ/mol
  + Solvation: -20 kJ/mol
  = Net: -30 kJ/mol
```

    - Amplitude: Neck linker moves ~10 nm

Measurement (time-resolved spectroscopy):
  - Ultrafast IR shows: Pi release → backbone change in <10 ps
  - Fluorescence: Conformational change complete in ~0.1-1 ms
  - Single-molecule: Force develops over ~1 ms
```

### State 3: Trailing head detachment

```
Trigger: Neck linker docking in LH

Mechanism:
  - Mechanical strain on TH increases
  - TH-MT interface destabilized
  - ADP release from TH
  - TH detaches

VIBRATIONAL:
  
Strain propagation:
  - LH conformational change creates tension in stalk
  - Tension = stretching of coiled-coil
  - Coiled-coil vibrational modes excited (0.01-0.1 THz)
  - Energy propagates to TH (~10 nm away)
  - Timescale: ~10-100 ns (speed of sound in protein ~2000 m/s)

TH response:
  - Strain increases amplitude of TH oscillations
  - TH-MT interface bonds weaken (reduced coupling)
  - ADP release facilitated (pocket opens more easily)
  - Detachment: TH breaks H-bonds with MT

This is forced decoupling:
  - Mechanical force weakens vibrational coupling
  - Pattern (TH-MT complex) destabilized
  - Decoherence occurs (dissociation)
```

### State 4: Diffusive search (TH finding next site)

```
TH now detached:
  - Tethered by stalk (coiled-coil)
  - But free to diffuse
  - Must find next MT binding site (+8 nm forward)

Brownian motion:
  - TH undergoes random walk
  - Constrained by stalk (acts like spring)
  - Biased forward (stalk under tension)

CYMATIC VIEW:
  
Detached head = Free oscillator
  - No boundary condition (unbound)
  - High amplitude thermal oscillations
  - Random phase (no coherence with MT)
  
Stalk = Spring coupler
  - Provides restoring force
  - Biases diffusion forward
  - Spring constant: ~0.1-1 pN/nm
  - Potential energy: U = ½k(x-x₀)² where x₀ = +8 nm
  
Diffusion dynamics:
  - Diffusion coefficient: D ~ 10⁻¹² m²/s (small protein in water)
  - Mean square displacement: ⟨x²⟩ = 2Dt
  - Time to move 8 nm: τ ~ x²/(2D) ~ 30 μs
  - But must also find correct orientation
  - Effective search time: ~100 μs to 1 ms

Binding:
  - When TH near MT binding site
  - Weak electrostatic attraction (~2 nm range)
  - Guides final approach
  - Binding occurs when geometry matches
  - Initial binding: Weak (can still dissociate)
```

### State 5: New leading head binds ATP

```
TH now weakly bound at position n+1:
  - Becomes new leading head
  - ATP binds (diffusion-limited, <0.1 ms)

ATP binding triggers:
  - Conformational change in new LH
  - Neck linker docks
  - Stalk straightens
  - Old LH (now TH) destabilized

Cycle repeats from State 1
```
